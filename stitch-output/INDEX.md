## 4. ASSISTANT STACK (16 telas)

- **Assistant Screen:** Chat principal com IA, VoiceButton e histórico

- **Assistant Settings:** Configurações de voz e auto-play
- **Script Generation Modal:** Input de tema e duração para gerar roteiro

- **Processing Video Loading:** Tela de loading com etapas de processamento
- **Video Final Preview:** Preview do vídeo editado com toggle antes/depois
- **Subtitles Customization Modal:** Customização de legendas (cor, tamanho, fonte)
- **Edit Script Modal:** Edição de roteiro com word counter
- **Saved Scripts Screen:** Lista de roteiros salvos
- **Choose Script Screen:** Seleção de roteiro para gravar
- **Teleprompter Settings:** Configurações de scroll e velocidade
